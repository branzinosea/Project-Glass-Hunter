  The Mission
[cite_start]Repurposing decommissioned military technology to detect invisible shrapnel in frontline triage units[cite: 1, 30].

 Hardware & Sanitization
* [cite_start]**Target:** Surplus 4.5-gen fighter radar array faces (e.g., AN/APG-63, F-16 SABR)[cite: 7, 9, 11].
* [cite_start]**The "Lobotomy":** Physically remove back-end processors and military FPGAs to bypass ITAR[cite: 16, 17]. 
* [cite_start]**The Result:** A high-performance, programmable RF emitter with no classified algorithms[cite: 20].

  How it Works
* [cite_start]**Dielectric Contrast:** Human tissue ($\epsilon \approx 80$) is transparent, while glass shrapnel ($\epsilon \approx 4 \sim 10$) reflects RF waves like a beacon[cite: 24, 25].
* [cite_start]**Resolution:** Achieves sub-millimeter 3D mapping using massive bandwidth in a Near-Field SAR configuration[cite: 26, 27].
* [cite_start]**Computation:** Processes data via commercial GPUs (Nvidia RTX) or Raspberry Pi clusters[cite: 28].

  Field Deployment
* [cite_start]**Power:** Throttled down to the milliwatt (mW) range—safe as a Wi-Fi router[cite: 32, 34].
* [cite_start]**Portability:** Runs on standard 220V AC or ambulance batteries[cite: 33].
* [cite_start]**Durability:** Conformal coating allows the unit to be scrubbed with soap and water to remove blood[cite: 37, 38].
